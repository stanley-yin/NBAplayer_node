# NBAplayer_node

1. Clone the repository
```
https://github.com/stanley-yin/NBAplayer_node.git
```

2. Install required dependencies

```
yarn install
```

3. setting MySQL connection

```
open connect-mysql.js , change user &　password
```

4. import players.json into MySQL database

```
node importData.js
```

5. start server

```
node index.js
```
